中医治疗系统图标生成器

使用说明：

  1. 点击下方的"生成图标"按钮
  2. 右键点击生成的图标，选择"另存为"
  3. 将图标保存到icons文件夹中
  4. 按照尺寸重命名文件：
    • 72x72.png → icon-72x72.png
    • 96x96.png → icon-96x96.png
    • 128x128.png → icon-128x128.png
    • 144x144.png → icon-144x144.png
    • 152x152.png → icon-152x152.png
    • 192x192.png → icon-192x192.png
    • 384x384.png → icon-384x384.png
    • 512x512.png → icon-512x512.png
